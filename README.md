# Rust Number Theory Exercises

Учебный проект с алгоритмическими задачами по теории чисел на Rust.

## Содержание

- работа с диапазонами и векторами;
- функции и условные выражения;
- итераторы;
- переборные численные алгоритмы;
- базовые правила владения данными.

## Технологии

Rust, Cargo.

## Запуск

```bash
cargo run
```
